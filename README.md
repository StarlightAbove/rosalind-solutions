# rosalind-solutions
Solutions I have wrote for Rosalind Bioinformatics problems, in C++, Python and R.

## About

This repository contains solutions for bioinformatics problems from [Rosalind](https://rosalind.info/), implemented in C++, Python, and R.

## Structure

Solutions are organized by programming language in their respective directories.

Each solution is depicted by the abbreviation used in the Rosalind tree.

## Getting Started

Clone the repository and navigate to the language-specific folder to view implementations and comments.
